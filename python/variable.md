# 변수

1. 숫자
2. 문자
3. 참/거짓



## 1. 숫자

```python 
number1 = 1 # number라는 변수에 정수를 넣을 수 있다.
number2 = -3 # 음수도 가능.
number3 = 3j+4 # 복소수도 넣을 수 있다.
```

## 2. 문자

> 파이썬에서 큰따옴표와 작은따옴표는 동일하다.

```python
string1 = "파이썬" # 파이썬에서는 큰따옴표 안에 문자열을 넣는다.
```

## 3. 참/거짓

```python
my_bool = True # 시작은 대문자로 적어야한다.
```
